# Pac-Man Clone in Go

Develop a classic Pac-Man clone in Go using the Ebitengine game library, leveraging its vector graphics capabilities.

## Key Features:

*   **Platform:** Go with Ebitengine (v2.9.4+).
*   **Graphics:** Pure vector rendering for all game elements.
*   **Maze:** Grid-based level design with walls and collectible dots.
*   **Player:** Animated yellow Pac-Man, controlled by arrow keys.
*   **Ghosts:** Four uniquely colored, AI-controlled ghosts with basic random movement.
*   **Gameplay:**
    *   Responsive, grid-aligned movement for both Pac-Man and ghosts.
    *   Collect dots for points.
    *   Game ends upon collision between Pac-Man and a ghost.
*   **UI:** Real-time score display and a "Game Over" screen.
